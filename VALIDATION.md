## ✅ 기능 검증

### 카테고리 전환
- ✅ "길이" 탭 클릭 → 길이 입력 표시
- ✅ "무게" 탭 클릭 → 무게 입력 표시
- ✅ "온도" 탭 클릭 → 온도 입력 표시
- ✅ "부피" 탭 클릭 → 부피 입력 표시
- ✅ "넓이" 탭 클릭 → 넓이 입력 표시
- ✅ "속도" 탭 클릭 → 속도 입력 표시

### 실시간 변환
- ✅ 길이: 평 → ㎡ 변환 (1평 = 3.306㎡)
- ✅ 무게: kg → lb 변환 (1kg = 2.205lb)
- ✅ 온도: ℃ → ℉ 변환 (0℃ = 32℉)
- ✅ 부피: L → gal 변환 (1L = 0.264gal)
- ✅ 넓이: ㎡ → 평 변환 (1㎡ = 0.303평)
- ✅ 속도: km/h → mph 변환 (1km/h = 0.621mph)

### 스왑 버튼
- ✅ 두 입력값 교환

- ✅ 모든 카테고리에서 작동
